1. Cargar base de datos

Descarga la plantilla para capturar los datos con los encabezados correctos. El Excel solo debe incluir nombre, sexo/género y promedio del examen; la carrera se asigna automáticamente con la configuración de grupos.

2. Configurar grupos por carrera

Agrega una fila por carrera. Puedes indicar cuántos grupos habrá en matutino y vespertino; la capacidad predeterminada es 50 alumnos.

Carrera Grupos matutino Grupos vespertino Capacidad

3. Generar resultado

El sistema asigna la carrera y reparte grupos, buscando equilibrar hombres/mujeres, mejores promedios, grupos y turnos.
