1. Cargar base de datos

Carga el Excel de secretaría con los encabezados nombre, carrera, sexo/género y, si está disponible, promedio del examen. La carrera ya no se asigna aquí: el sistema respeta la carrera indicada en el archivo y solo equilibra los grupos dentro de cada carrera.

2. Configurar grupos por carrera

Agrega una fila por cada carrera que venga en el Excel para indicar cuántos grupos habrá en matutino y vespertino; también puedes escribir los nombres de los grupos separados por coma (por ejemplo: 1A, 1B, 1C). La capacidad predeterminada es 50 alumnos. Si alguna carrera no está configurada, se crearán grupos automáticos para esa carrera.

3. Generar resultado

El sistema conserva la carrera asignada por secretaría y reparte grupos usando los nombres que escribiste; si dejas nombres vacíos, usará M1, V1 o AUTO como respaldo.
