Asignación equilibrada de grupos

Dirección y Control Escolar

1. Cargar base de datos

El Excel debe incluir encabezados para carrera, nombre, sexo/género y promedio del examen.

2. Configurar grupos por carrera

Agrega una fila por carrera. Puedes indicar cuántos grupos habrá en matutino y vespertino; la capacidad predeterminada es 50 alumnos.

Carrera Grupos matutino Grupos vespertino Capacidad

3. Generar resultado

El sistema reparte por carrera, buscando equilibrar hombres/mujeres, mejores promedios, grupos y turnos.
